Frequently Asked Questions

Is the toolkit really free?

Yes! The M365 Security Toolkit is 100% free and open-source under the MIT License. You can use it for personal or commercial purposes without any cost. Professional services and custom development are available for organizations needing additional support.

What M365 licenses do I need?

The toolkit works with Microsoft 365 Business Basic, Business Standard, Business Premium, and Enterprise (E3/E5) plans. Some advanced controls require E3/E5 features like Microsoft Defender for Office 365.

Can I use this for multiple tenants?

Absolutely! The toolkit supports multi-tenant audits, making it perfect for MSPs managing multiple client M365 environments. The GitHub Action includes batch configuration for parallel tenant audits.

How often should I run audits?

We recommend monthly audits as a baseline. High-security environments may want weekly or even daily automated checks. The GitHub Action can be scheduled using cron expressions for automated recurring audits.

Does this modify my M365 tenant?

By default, the toolkit is read-only and will not make any changes. Remediation features are optional and include a -WhatIf preview mode to review changes before applying them. All remediation actions are logged for audit purposes.

Can I customize the controls?

Yes! The PowerShell module is fully customizable. You can add custom controls, modify existing ones, or disable checks that don't apply to your organization. Documentation on custom control development is available.
