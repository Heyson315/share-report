Privacy Policy

Last updated: December 11, 2024

Introduction

Rahman Finance and Accounting P.L.LLC ("we", "our", "us") respects your privacy. This Privacy Policy explains how we collect, use, and protect information when you use the M365 Security Toolkit.

Information We Collect

Open Source Toolkit

When you use the open-source M365 Security Toolkit:

  • We do NOT collect any personal information
  • All audit data remains in your M365 tenant and local environment
  • No telemetry or usage tracking is performed
  • GitHub repository interactions follow GitHub's privacy policy

Website Analytics

This website may use:

  • GitHub Pages hosting (subject to GitHub's privacy policy)
  • Basic web analytics for page views and traffic patterns
  • No personally identifiable information is stored

Data Storage and Security

The M365 Security Toolkit:

  • Stores audit results locally on your system or GitHub repository
  • Does not transmit data to external servers
  • Follows secure coding practices documented in our repository
  • Supports encryption and secure credential storage

Third-Party Services

The toolkit may interact with:

  • Microsoft 365 APIs (subject to Microsoft's privacy policy)
  • GitHub Actions (subject to GitHub's privacy policy)
  • Optional integrations you configure (SharePoint, QuickBooks, etc.)

Your Rights

You have the right to:

  • Use the toolkit without providing any personal information
  • Control all data generated by the toolkit
  • Delete any audit data stored locally
  • Fork and modify the source code under the MIT License

Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page with a new "Last updated" date.
